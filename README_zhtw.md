### 前言

當你心中充滿著想要製作文字類型遊戲、亦或是小品解謎遊戲時， <br>
是否會因為不會寫程式或是不知道要利用甚麼框架寫遊戲而感到苦惱？ <br>
這時，互動式文字引擎就可以派上用場，解決這類型的所有問題啦！ <br>
互動式文字引擎可以藉由簡單的編寫劇本檔，建立一個具有webserver功能的文字遊戲。 <br>
它不只是一個簡簡單單的前端生成引擎，同時具備許多酷炫的功能以及前後端交互的元素。 <br>
<br>
